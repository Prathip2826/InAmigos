## ✨ Features

- ✅ Fully **responsive** — works on mobile, tablet & desktop
- ✅ **Scroll-triggered animations** (fade-up on scroll)
- ✅ **Hover effects** on all project & impact cards
- ✅ **No frameworks** — pure HTML, CSS, and vanilla JavaScript
- ✅ Google Fonts: **Playfair Display** + **DM Sans**
- ✅ Real data from [inamigosfoundation.org.in](https://inamigosfoundation.org.in)
- ✅ All 6 NGO projects covered: SEVA, Bachpanshala, Udaan, Prakriti, Jeev, VIKAS

## 🛠️ Local Setup

No installation needed. Just:

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/inamigos-awareness.git

# Open in browser
cd inamigos-awareness
open index.html        # macOS
# OR
start index.html       # Windows
# OR just drag index.html into any browser
```
